# 🏅 Player Insights Dashboard (Blazor + AI)

A professional **Player Insights Dashboard** built using **Blazor** that provides AI-powered analytics and performance insights for sports players.  
The application consumes a backend API to fetch player data and generate intelligent insights in a clean, corporate-style UI.

---

## 🚀 Features

- ✅ Professional corporate UI design  
- ✅ Responsive dashboard layout  
- ✅ AI-powered player insights  
- ✅ Clean card-based analytics view  
- ✅ Animated loading indicator  
- ✅ Smooth fade-in animations  
- ✅ API-driven architecture  

---

## 🧰 Tech Stack

### Frontend
- **Blazor (.NET 8)**
- **Bootstrap**
- **CSS (Custom Corporate Styling)**
- **Font Awesome (Icons)**

### Backend (Integrated)
- **.NET Web API**
- **FastAPI (Python – AI Processing Layer)**
- **OpenAI API (LLM / NLP for Insights)**

---


## 🖥️ UI Overview

### 🔹 Input Section
- Player ID input
- Sport input
- “Get AI Insight” action button with loading animation

### 🔹 Result Section
- Player name & sport
- Average score
- Key performance highlight
- AI-generated detailed report

---

## 🔌 API Integration

The Blazor app communicates with a backend API using `SportsApiService`.


---

## ▶️ How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Open the solution in **Visual Studio**

3. Restore dependencies

```bash
dotnet restore
```

4. Run the application

```bash
dotnet run
```

5. Navigate to:

```
https://localhost:<port>/player-insights
```
